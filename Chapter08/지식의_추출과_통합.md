# 지식의 추출과 통합 - LlamaIndex를 이용한 검색증강생성 구현

검색증강생성 = 정보 검색과 텍스트 생성을 결합한 인공지능 기술

검색증강생성은 2가지 주요 부분으로 구분됨
1. 정보 검색
- 대규모 데이터 집합에서 관련 정보를 검색
- 방대한 텍스트 데이터를 포함하며, 문서에 한정하지 않고 이미지, 코드 등 많은 형식을 포함함
2. 텍스트 생성
- 정보를 기반으로 응답을 생성함

사용자 요청 -> 요청을 기반으로 문맥 정보 추출 -> llm으로 적합한 응답 생성

응답 정확도
검색증강생성 < 파인튜닝
큰 차이가 나진 않으나, 파인튜닝 비용이 큼
따라서 검색증강생성이 llm의 상용화 과정에서 비용 면에서 더 효율적인 선택임

기술적 관점에서 검색 부분의 파이프라인
데이터 -> 로드 -> 변환 -> 임베딩 -> 저장 -> 검색

사용자 관점에서 본 검색증강생성 과정
지식 기반 데이터베이스 -> 색인 -> llm -> 사용자

검색증강 생성에서는 검색 단계와 생성 단계가 서로 보완적인 역할을 함
검색 단계는 사용자 요청과 직접 관련된 정보를 제공하고, 생성 단계는 이 정보를 활용해 일관성 있고 정확하며 더 상세한 응답을 생성함
