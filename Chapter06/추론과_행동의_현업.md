## 추론과 행동의 협업 - LangChain의 ReAct기반 체계를 이용한 자동 가격 설정 구현

ReAct 기반 체계 복습
상황 : 꽃가게를 운영하다 보면 날씨에 따라 가격이 바뀌는 상황이 마주할때 꽃의 가격을 어떻게 결정할 것인가
처리과정
1. 사고 : 이 상황을 어떻게 처리할 지 생각함
2. 행동 : 검색엔진을 통해 오늘의 꽃 원가를 확인함
3. 관찰 : 꽃의 입고 가격을 예측하고, 그 가격 수준을 확인함
4. 사고 : 마진으로 얼마를 추가할지 결정함
5. 행동 : 판매 가격을 정함

ReAct는 추론과 행동을 조합하여 llm이 과제를 해결하는 경로를 생성함
즉, 환경을 관찰하고, 생각하고, 관찰-추론-행동 순서로 진행됨

---
langchain에서 react 에이전트 구현하기

langchain에서 에이전트 사용시 4가지 요소가 중요
1. llm : 논리적 엔진을 제공하며 에측을 생성하고 입력을 처리하는 역할
2. 프롬프트 : 모델을 안내하며 추론 기반 체계를 형성
3. 외부 도구 : 데이터 정리 도구, 검색엔진, 애플리케이션 등을 포함
4. 에이전트 실행기 : 적합한 외부 도구를 호출하고 전체 과정을 관리

langchain은 ReAct기반 체계를 완벽하게 패키징하고 구현했기에 llm에 큰 자율성을 부여할 수 있음
langchain의 에이전트는 일반적으로 llm과 프롬프트를 통해 구동됨
프롬프트는 에이전트가 특정한 방식으로 응답하도록 하여 성격을 결정하고, 더 많은 작업 유형에 대한 맥락을 통해 작업의 배경을 제공함
